# github-stats

## Software stack
- Java 17
- Spring Boot 2.7.5
- Apache Maven 3.8.6

## Build, test and run with Maven

```sh
# compile the code
./mvnw compile

# run tests
./mvnw test

# run app
./mvnw spring-boot:run

# package to a jar
./mvnw package

# run jar
java -jar target/github-stats-*.jar
```
